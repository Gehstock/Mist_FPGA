Midway 8080 Arcade Platform for Mist FPGA

- with OSD
- Keyboard/Joystick Support
- 15kHz RGB/VGA/YPbPr Supported

Controls:

--   ESC     or Button C     : Coin
--   1, 2    or Button Start : Start
--   SPACE   or Button A     : Fire
--   LEFT, RIGHT             : Move
